Review components are essential for showcasing user feedback, ratings, and testimonials in any application. Building them with composability, theming, and animation in mind ensures a seamless and engaging user experience.

I developed a set of review components that are highly customizable and animated using Framer Motion. These components leverage React's context API for state management, making them both flexible and easy to integrate.

## Installation

To integrate the review components into your project, follow the steps below.

## Structure

The review component library is composed of the following parts:

- **Contexts**: Manage shared state across review components.
- **Hooks**: Provide easy access to context values.
- **Animation Configurations**: Define default animations for components.
- **Motion Components**: Enhanced components with animation capabilities.
- **Review Components**: The main components used to build review cards and sections.

## Usage

Here's how to integrate and use the review components in your application.

### Basic Review Section

Start by creating a review section that will contain multiple review cards.

```tsx showLineNumbers title="components/ReviewSectionDemo.tsx"
import React from "react"

import {
  PKSReviewCard,
  PKSReviewCardAuthor,
  PKSReviewCardDescription,
  PKSReviewCardHeader,
  PKSReviewCardRating,
  PKSReviewCardTitle,
  ReviewSection,
  ReviewSectionDescription,
  ReviewSectionHeader,
  ReviewSectionReviews,
} from "@/components/ui/reviews"

export function ReviewSectionDemo() {
  return (
    <ReviewSection>
      <ReviewSectionHeader>
        <h2>What Our Customers Say</h2>
      </ReviewSectionHeader>
      <ReviewSectionDescription>
        Hear from users who have experienced the benefits of our product.
      </ReviewSectionDescription>
      <ReviewSectionReviews>
        <PKSReviewCard rating={5}>
          <PKSReviewCardHeader>
            <PKSReviewCardTitle>Excellent Service</PKSReviewCardTitle>
          </PKSReviewCardHeader>
          <PKSReviewCardDescription>
            The customer service was outstanding. Highly recommend!
          </PKSReviewCardDescription>
          <PKSReviewCardAuthor
            name="Jane Doe"
            title="Product Manager"
            company="TechCorp"
            avatar="/avatars/jane.jpg"
          />
        </PKSReviewCard>
        {/* Add more PKSReviewCard components as needed */}
      </ReviewSectionReviews>
    </ReviewSection>
  )
}
```

## Components

### ReviewSection

The `ReviewSection` component serves as a container for multiple review cards. It handles layout and animation configurations.

#### Props

| Name        | Type                           | Description                                      |
| ----------- | ------------------------------ | ------------------------------------------------ |
| `animation` | `ReviewSectionAnimationConfig` | Custom animation configurations for the section. |

#### Example

```tsx showLineNumbers title="components/CustomReviewSection.tsx"
import React from "react"

import {
  PKSReviewCard,
  ReviewSection,
  ReviewSectionHeader,
  ReviewSectionReviews,
} from "@/components/ui/reviews"

export function CustomReviewSection() {
  return (
    <ReviewSection animation={{ disabled: false }}>
      <ReviewSectionHeader>
        <h2>User Feedback</h2>
      </ReviewSectionHeader>
      <ReviewSectionReviews>
        <PKSReviewCard rating={5}>{/* Review card content */}</PKSReviewCard>
        {/* More review cards */}
      </ReviewSectionReviews>
    </ReviewSection>
  )
}
```

### PKSReviewCard

A card component for displaying individual reviews, including rating, description, and author information.

#### Props

| Name        | Type              | Description                                |
| ----------- | ----------------- | ------------------------------------------ |
| `rating`    | `number`          | The rating value for the review (e.g., 4). |
| `maxRating` | `number`          | The maximum rating value (default is 5).   |
| `children`  | `React.ReactNode` | The content of the review card.            |
| `className` | `string`          | Additional CSS classes for styling.        |

#### Example

```tsx showLineNumbers title="components/SimpleReviewCard.tsx"
import React from "react"

import {
  PKSReviewCard,
  PKSReviewCardAuthor,
  PKSReviewCardDescription,
  PKSReviewCardHeader,
  PKSReviewCardTitle,
} from "@/components/ui/reviews"

export function SimpleReviewCard() {
  return (
    <PKSReviewCard rating={5}>
      <PKSReviewCardHeader>
        <PKSReviewCardTitle>Outstanding Experience</PKSReviewCardTitle>
      </PKSReviewCardHeader>
      <PKSReviewCardDescription>
        I had an outstanding experience with the support team. Highly
        recommended!
      </PKSReviewCardDescription>
      <PKSReviewCardAuthor
        name="Alice Johnson"
        title="Designer"
        company="CreativeStudio"
        avatar="/avatars/alice.jpg"
      />
    </PKSReviewCard>
  )
}
```

### Hooks

#### useReviewCard

Accesses the `ReviewCard` context. Must be used within a `PKSReviewCard`.

##### Usage

```tsx showLineNumbers
import { useReviewCard } from "@/components/ui/reviews"

function CustomComponent() {
  const { rating, maxRating, isWithinFooter } = useReviewCard()

  return (
    <div>
      Rating: {rating} / {maxRating}
    </div>
  )
}
```

#### useReviewSection

Accesses the `ReviewSection` context. Must be used within a `ReviewSection`.

##### Usage

```tsx showLineNumbers
import { useReviewSection } from "@/components/ui/reviews"

function CustomSectionComponent() {
  const { hasFooter } = useReviewSection()

  return hasFooter ? <div>Footer is present</div> : null
}
```

#### useReviewSectionReviews

Accesses the `ReviewSectionReviews` context. Must be used within a `ReviewSectionReviews`.

##### Usage

```tsx showLineNumbers
import { useReviewSectionReviews } from "@/components/ui/reviews"

function CustomReviewsComponent() {
  const { isWithinReviewSectionReviews } = useReviewSectionReviews()

  return isWithinReviewSectionReviews ? <div>Inside Reviews</div> : null
}
```

## Styling

Customize the review components further using utility classes or by extending their styles.

  You can style components based on their context states using data attributes.

  ```tsx
  <PKSReviewCard className="data-[isWithinFooter=true]:bg-gray-100">
    {/* Review card content */}
  ```

  ```tsx
  <PKSReviewCard className="shadow-lg hover:shadow-xl transition-shadow duration-300">
  ```

## Advanced Customization

### Custom Animation Configurations

Override default animations by providing custom configurations to `ReviewSection` and `ReviewSectionReviews`.

```tsx showLineNumbers title="components/CustomAnimatedReviewSection.tsx"
import React from "react"

import {
  PKSReviewCard,
  ReviewSection,
  ReviewSectionHeader,
  ReviewSectionReviews,
} from "@/components/ui/reviews"

const customAnimationConfig = {
  container: {
    initial: { opacity: 0 },
    animate: { opacity: 1, transition: { duration: 1 } },
  },
  header: {
    initial: { x: -100 },
    animate: { x: 0, transition: { type: "spring", stiffness: 50 } },
  },
}

export function CustomAnimatedReviewSection() {
  return (
    <ReviewSection animation={customAnimationConfig}>
      <ReviewSectionHeader>
        <h2>Customer Reviews</h2>
      </ReviewSectionHeader>
      <ReviewSectionReviews>
        <PKSReviewCard rating={5}>{/* Review card content */}</PKSReviewCard>
        {/* More review cards */}
      </ReviewSectionReviews>
    </ReviewSection>
  )
}
```

### Integrating with State Management Libraries

Use state management libraries like Redux or Zustand in conjunction with the review components for more complex state scenarios.

```tsx showLineNumbers title="components/ReduxReviewSection.tsx"
import React from "react"
import { useSelector } from "react-redux"

import {
  PKSReviewCard,
  ReviewSection,
  ReviewSectionReviews,
} from "@/components/ui/reviews"

export function ReduxReviewSection() {
  const reviews = useSelector((state) => state.reviews.list)

  return (
    <ReviewSection>
      <ReviewSectionReviews>
        {reviews.map((review) => (
          <PKSReviewCard key={review.id} rating={review.rating}>
            {/* Review card content */}
          </PKSReviewCard>
        ))}
      </ReviewSectionReviews>
    </ReviewSection>
  )
}
```
